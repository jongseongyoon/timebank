## 1. TimePay란 무엇인가?

**TimePay**는 광주광역시 서구 주민들이 서로의 시간과 재능을 나누는 **타임뱅크(Time Bank) 플랫폼**입니다.

### 핵심 개념 — TC (Time Credit)

| 개념 | 설명 |
|------|------|
| **TC** | 시간 화폐. 1시간 봉사 = 1 TC 적립 |
| **적립** | 이웃에게 서비스를 제공하면 TC가 쌓입니다 |
| **소진** | 내가 서비스를 받을 때 TC를 씁니다 |
| **공정 교환** | 누구의 1시간이든 동일한 가치로 인정됩니다 |

> 예시: 홍길동 씨가 이웃의 장보기를 2시간 도와주면 **2.0 TC** 적립.  
> 나중에 다른 이웃에게 말벗 서비스 1시간을 받으면 **1.0 TC** 차감.

### 역할 구분

| 역할 | 설명 |
|------|------|
| **회원 (일반)** | 서비스 제공·수혜, 커뮤니티 활동 |
| **코디네이터** | 거래 기록·승인, 회원 매칭, 돌봄 패키지 관리 |
| **관리자** | 전체 시스템 관리, TC 배분, 통계 조회 |

## 2. 회원가입 · 로그인

### 2-1. 회원가입

1. **https://timebank-mocha.vercel.app/register** 접속
2. 아래 정보를 입력합니다

| 항목 | 필수 | 설명 |
|------|------|------|
| 이름 | ✅ | 실명 입력 |
| 전화번호 | ✅ | 010-0000-0000 형식 |
| 비밀번호 | ✅ | 8자 이상 |
| 출생연도 | ✅ | 숫자 4자리 (예: 1975) |
| 동 | ✅ | 거주 동 선택 |
| 이메일 | 선택 | 소셜 로그인용 |

3. **회원가입** 버튼 클릭 → 자동으로 로그인 화면으로 이동

> ⚠️ 전화번호는 나중에 TC 거래 시 본인 확인에 사용됩니다. 정확히 입력하세요.

### 2-2. 로그인

1. **https://timebank-mocha.vercel.app/login** 접속
2. 전화번호 + 비밀번호 입력 후 **로그인**

### 2-3. 앱처럼 설치하기 (PWA)

TimePay는 스마트폰에 앱처럼 설치할 수 있습니다.

**Android (크롬)**
1. 브라우저 하단의 **"TimePay 앱 설치"** 배너가 나타나면 탭
2. 또는 크롬 메뉴(⋮) → **"홈 화면에 추가"**

**iPhone (사파리)**
1. 하단 공유 버튼(□↑) 탭
2. **"홈 화면에 추가"** 선택
3. **추가** 탭

설치 후 홈 화면에서 TimePay 아이콘으로 바로 실행됩니다.

## 4. TC 지갑

**홈 → TC 잔액 카드** 또는 **내정보 → 지갑**

### 확인 가능한 정보

- **현재 잔액**: 지금 사용할 수 있는 TC
- **총 적립**: 지금까지 벌어들인 TC 합계
- **총 소진**: 지금까지 사용한 TC 합계
- **만료일**: TC 유효기간 (설정된 경우)

### 거래 내역 보기

**내정보 → 거래 내역** 또는 홈의 **전체보기** 클릭

각 거래에서 확인 가능:
- 상대방 이름 (이니셜 처리)
- 서비스 종류
- TC 금액 (+적립 / -소진)
- 거래 상태 (승인 대기 / 완료 / 취소 등)

## 5. 내 QR 코드

QR 코드는 **본인 확인 수단**으로, 거래 시 상대방에게 보여주거나 스캔합니다.

### QR 코드 발급받기

1. **하단 바 → QR(중앙 버튼)** 또는 **내정보 → 내 QR 코드**
2. QR 코드가 없는 경우 → **"QR 코드 생성하기"** 버튼 탭
3. 생성 완료 후 화면에 QR 코드가 표시됩니다

### QR 저장 · 공유

| 버튼 | 기능 |
|------|------|
| **QR 저장하기** | 이미지로 갤러리에 저장 |
| **QR 공유하기** | 카카오톡·문자 등으로 공유 (지원 기기만) |

> 💡 QR 코드를 출력해서 지갑에 넣어 두면 오프라인에서도 편리합니다.

## 6. QR 스캔 거래

하단 중앙의 파란 **QR 버튼**을 탭하면 카메라가 열립니다.

### 6-1. TC 직접 이체

이웃에게 TC를 바로 보낼 때 사용합니다.

1. **QR 버튼** → 상대방 QR 코드 스캔
2. 상대방 정보 확인 (이름, 동, 별점)
3. **"TC 이체"** 선택
4. 이체할 TC 금액 입력 (0.5 단위)
5. 메모 입력 (선택)
6. **"이체 확인"** 버튼 탭 → 즉시 처리

### 6-2. 서비스 시작 (QR 체크인)

서비스를 시작할 때 시간을 기록합니다.

1. **QR 버튼** → 서비스 수혜자의 QR 스캔
2. **"서비스 시작"** 선택
3. 서비스 종류 선택 (카테고리)
4. **"시작"** 버튼 탭 → 타이머 시작

### 6-3. 서비스 종료 (QR 체크아웃)

서비스가 끝나면 종료를 기록합니다.

1. **QR 버튼** → 같은 수혜자의 QR 재스캔
2. 진행 중인 서비스 확인 → **"서비스 종료"**
3. 자동으로 소요 시간 계산 → TC 정산
4. 수혜자가 별점(1~5점) 평가 → 완료

> ⏱️ TC 계산: 실제 소요 시간(분) ÷ 60 = TC  
> (최소 15분, 이후 1분 단위)

### 별점 자동 팝업

서비스 완료 후 24시간 이내에 앱을 열면 **별점 평가 팝업**이 자동으로 뜹니다.  
1~5점 + 후기 텍스트(선택)를 입력하여 서로를 평가하세요.

## 7. 서비스 찾기 · 신청

**하단 바 → 서비스** 또는 홈의 **"서비스 요청하기"**

### 검색 · 필터 사용법

| 필터 | 설명 |
|------|------|
| 🔍 검색창 | 서비스명, 제공자 이름으로 검색 |
| 카테고리 | 이동지원, 장보기, 말벗 등 선택 |
| 지역 | 동 이름으로 필터링 |
| 단체만 | 기관·단체 서비스만 표시 |
| 📍 내 주변 순 | 현재 위치 기준 가까운 서비스 순 정렬 |

### 내 주변 순 사용법

1. **"내 주변 순"** 버튼 탭
2. 브라우저에서 위치 권한 허용
3. 자동으로 거리 순 정렬 + 카드에 거리 표시 (예: `0.8km`)

### 서비스 신청하기

1. 원하는 서비스 카드 클릭
2. 상세 내용 확인 (카테고리, TC/시간, 가능 지역·요일·시간)
3. **전화로 먼저 문의**: 제공자 전화번호 버튼 탭 → 바로 전화 연결
4. **신청하기** 버튼 클릭
5. 희망 날짜/시간, 소요 시간, 요청 사항 입력
6. **신청하기** → 코디네이터가 검토 후 연결

## 8. 서비스 등록

내가 제공할 수 있는 서비스를 등록합니다.

**하단 바 → 서비스 → (홈의) "활동 등록하기"**

### 등록 순서

1. **서비스 위치** (선택): "현재 위치 감지" 버튼 탭 → 내 주변 검색에 노출
2. **서비스 유형**: 카테고리 선택 (15가지)
3. **기본 정보**: 제목, 상세 설명 입력
4. **제공 가능 지역**: 동 선택 (복수 선택 가능)
5. **가능 요일**: 월~일 선택
6. **가능 시간**: 시작~종료 시간 설정
7. **서비스 등록하기** 버튼 클릭

> 💡 카테고리별 TC 단가는 시스템에서 자동 계산됩니다.  
> 이동지원·의료동행: 1.5 TC/h | 교육·법률상담: 2.0 TC/h | 나머지: 1.0 TC/h

## 9. 만보기 — TC 적립 걷기

**홈 화면 → 초록 만보기 카드** 또는 직접 `/walk` 접속

매일 10,000보를 걸으면 **0.5 TC**가 자동으로 적립됩니다.

### 사용 방법

1. **"걷기 측정 시작"** 버튼 탭
   - iPhone: 센서 권한 허용 팝업 → **허용** 클릭
2. 스마트폰을 주머니에 넣거나 손에 들고 걷기
3. 원형 게이지가 채워지는 것을 확인
4. 측정을 멈출 때 **"측정 중지 및 저장"** 탭 → 서버에 저장
5. 10,000보 달성 시 🎉 축하 화면 + 0.5 TC 자동 적립

### 주의 사항

- 매일 **자정(00:00)**에 걸음 수 초기화
- 하루 최대 1회 TC 지급 (10,000보 이상이어도 0.5 TC)
- **중지 버튼**을 눌러야 최종 저장됩니다
- 앱을 닫으면 측정이 중단됩니다

## 10. 커뮤니티 게시판

**하단 바 → 커뮤니티**

이웃들과 소식을 나누고, 타임뱅크 후기를 공유하세요.

### 게시판 종류

| 카테고리 | 설명 |
|----------|------|
| 📌 공지사항 | 운영진 공식 공지 (관리자만 작성) |
| 💬 자유게시판 | 자유로운 대화 |
| ⭐ 서비스 후기 | 거래 경험 공유 |
| ❓ 질문 | 시스템·서비스 문의 |

### 글 작성하기

1. 오른쪽 아래 **"글쓰기"** 버튼
2. 카테고리 선택 → 제목 → 내용 입력
3. **"등록"** 버튼

### 좋아요 · 댓글

- 게시글 하단 **❤️** 버튼 → 좋아요 토글
- **댓글 입력란**에 작성 후 **"등록"**

### 게시글 삭제

- 본인이 작성한 글: 상세 화면 → **삭제** 버튼
- 관리자는 모든 게시글 삭제 가능

## 11. 코디네이터 기능

> 코디네이터 권한이 있는 계정에서만 사용 가능합니다.  
> `/coordinator` 화면으로 이동하거나 사이드바에서 접근합니다.

### 주요 기능 목록

| 메뉴 | 설명 |
|------|------|
| **대시보드** | 오늘 처리할 거래, 매칭 현황 |
| **거래 기록** | 직접 거래 기록 (QR 없이 수동 입력) |
| **승인 대기** | PENDING 상태 거래 승인·거절 |
| **서비스 매칭** | 요청된 서비스를 제공자와 연결 |
| **회원 관리** | 담당 지역 회원 TC 잔액 조회 |
| **돌봄 패키지** | 장기 돌봄 프로그램 생성·관리 |
| **오프라인 동기화** | 종이 장부 → 시스템 일괄 입력 |

### 거래 직접 기록하기

1. **코디네이터 → 거래 기록**
2. 제공자 선택 (전화번호 검색)
3. 수혜자 선택
4. 서비스 종류, 날짜, 소요 시간 입력
5. 검증 방법 선택 (앱QR / 전화 / 종이 / 코디네이터)
6. **저장** → 자동으로 TC 정산

### 서비스 매칭하기

1. **코디네이터 → 서비스 매칭**
2. OPEN 상태의 요청 확인
3. 적합한 제공자 선택 → **매칭**
4. 제공자에게 연락 후 일정 조율

## 12. 관리자 기능

> 관리자(ADMIN) 권한 계정에서만 사용 가능합니다.  
> `/admin` 으로 접속합니다.

### 주요 메뉴

| 메뉴 | 설명 |
|------|------|
| **대시보드** | 전체 통계 (회원 수, TC 유통량, 활성 거래) |
| **회원 관리** | 회원 목록 조회, TC 직접 배분 |
| **TC 배분** | 개인 TC 수동 지급 |
| **일괄 배분** | 엑셀 파일로 여러 명에게 한 번에 TC 지급 |
| **거래 관리** | 거래 내역 수정·삭제, TC 자동 복구 |
| **기금 현황** | 지불준비금 TC/현금 비율 확인 |
| **통계·보고서** | 동별 현황, TC 흐름, 엑셀 내보내기 |
| **알림 발송** | 전체·동별·개인 브라우저 알림 발송 |
| **시스템 설정** | TC 기본 단가, 만료일 등 정책 설정 |

### TC 개인 배분

1. **관리자 → TC 배분**
2. 회원 이름 또는 전화번호 검색
3. TC 금액 입력 (0.5 단위, 최대 50 TC)
4. 지급 사유 입력
5. **TC 배분** 버튼 → 즉시 지갑에 반영

### 엑셀 일괄 배분

1. **관리자 → 일괄 배분**
2. **"양식 다운로드"** → 엑셀 파일 열기
3. A열: 전화번호, B열: 이름, C열: TC 금액, D열: 사유 입력
4. 파일 업로드 → 미리보기 확인
5. **"일괄 처리"** → 결과 확인 (성공/실패 표시)

> 💡 엑셀 양식 예시:
> ```
> 전화번호        이름    TC금액  사유
> 010-1234-5678  홍길동  5.0    자원봉사 참여 보상
> 010-9876-5432  김철수  2.5    지역 행사 참여
> ```

### 거래 삭제 (TC 자동 복구)

1. **관리자 → 거래 관리** → 해당 거래 찾기
2. **삭제** 버튼
3. 확인 → APPROVED 상태였던 거래는 **TC 자동 복구** (제공자 차감, 수혜자 반환)

## 14. 자주 묻는 질문 (FAQ)

### Q. TC가 차감됐는데 서비스를 받지 못했어요
→ 코디네이터 또는 관리자에게 문의하세요. 관리자는 거래를 취소하고 TC를 복구할 수 있습니다.

### Q. QR 코드가 없다고 나와요
→ **내 QR 코드 화면** → **"QR 코드 생성하기"** 버튼을 탭하면 즉시 발급됩니다. 기존 가입자는 이 과정이 필요합니다.

### Q. 만보기가 걸음을 세지 않아요
→ iPhone은 반드시 **"허용"** 팝업을 눌러 센서 권한을 주어야 합니다.  
→ 스마트폰을 흔들지 않고 자연스럽게 걷는 동작에서 감지됩니다.  
→ 걷는 속도가 너무 느리거나 너무 빠른 경우 일부 누락될 수 있습니다.

### Q. 서비스 거래 후 별점 팝업이 안 떠요
→ 거래 완료(APPROVED) 후 **24시간 이내**에 앱을 열어야 합니다.  
→ 이미 평가했거나 24시간이 지난 경우 팝업이 표시되지 않습니다.

### Q. "내 주변 순" 버튼이 작동하지 않아요
→ 브라우저에서 위치 권한을 **허용**으로 설정해야 합니다.  
→ iPhone: 설정 → Safari → 위치 → 허용  
→ Android: 설정 → 앱 → Chrome → 권한 → 위치 → 허용

### Q. TC는 언제 만료되나요?
→ 관리자가 설정한 만료일이 있는 경우, 홈 화면 TC 잔액 카드에서 확인할 수 있습니다.  
→ 만료일이 없는 경우 "무기한"으로 표시됩니다.

### Q. 비밀번호를 잊어버렸어요
→ 현재 버전에서는 관리자에게 직접 문의하여 비밀번호 초기화를 요청하세요.

### Q. 코디네이터/관리자 권한을 받으려면?
→ 관리자에게 연락하여 권한 부여를 요청하세요.
